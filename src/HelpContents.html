The Need for Bank Simulator

Bank Simulator is designed to simulate the queue of the waiting line in a bank. it will determine when the teller is free and then it will assign the next customer on line to that teller.

Saving Files

Bank Simulator Inc. developed the software with the saving capability so that users can save the simulate this software at their own time.

Opening files

You can open a saved file by going to File Menu and choosing open. You will then given a chance to locate the saved file so that you can continue playing the game.

Printing Results

In Simulator, you have the option to print the the results onto a paper so that you can review the results in your own time.

Progress Reports

Bank Simulator Inc. offers the conveneience of checking the queue. As you simulate the software, we will calculate your progress as a graph and display the results clearly for you. To view your progress, simply go to View menu and choose Graph.. or use the shortCuts below.
